Docker Container

All of the configuration for my container is hosted at the file system locationÂ /opt/appdata/oxidized

I will also select to execute the Web Interface for Oxidized using its default port with isÂ tcp:8888

Using the follow command, we will grab the latest container version from Docker Hub, and call the containerÂ oxidizedÂ locally. Additionally, if the container should stop, I am providing the flag to instruct docker to always restart the service again.

Configuration
